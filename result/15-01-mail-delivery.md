# 15.1. Доставка почты

## Результат
- **Статус:** ⚠️ Требует проверки
- **Severity:** S2
- **Краткое заключение:** не подтверждено, что письма отправляются через очереди; драйвер SMTP/SES не документирован.

## Наблюдение
- В `config/mail.php` заданы значения по умолчанию (нужно проверить `.env` на проде).
- Почтовые уведомления (`app/Mail/*`) могут отправляться синхронно, т.к. очереди отключены.

## Доказательства
- `app/Mail/*.php`
- Отключенные очереди (см. `10.1`)

## Риски
- Задержка ответов API при синхронной отправке писем.
- Потеря писем при ошибках SMTP.

## Рекомендации
1. Использовать `Queueable`/`ShouldQueue` для всех писем.
2. Включить Horizon и настроить retry/backoff.
3. Документировать почтовый провайдер (SMTP/SES/Mailgun) и доступы.
