# 23.2. Критерии прохождения блоков

## Результат
- **Статус:** ✅ Определены
- **Severity:** S3
- **Краткое заключение:** критерии «пройдено/не пройдено» закреплены и использовались при оценке блоков.

## Критерии
- Все обязательные пункты блока выполнены.
- Нет находок уровня S0/S1.
- Собраны доказательства (коммиты/скриншоты/логи).
- Для «требует проверки» фиксируется план действий.

Любая находка S0/S1 автоматически переводит блок в статус «не пройден».

## Рекомендации
- Оставить критерии неизменными и применять в последующих ревизиях.
