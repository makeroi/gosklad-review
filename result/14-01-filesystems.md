# 14.1. Конфигурация дисков

## Результат
- **Статус:** ❌ Не пройдено
- **Severity:** S2
- **Краткое заключение:** используется локальный диск (`public`/`storage`) для всех файлов; отсутствует разделение приват/публич и внешнее хранилище.

## Наблюдение
- `config/filesystems.php` не содержит конфигурации S3/MinIO.
- Документы и изображения сохраняются в `public/`.
- Резервные копии также хранятся локально.

## Доказательства
- `config/filesystems.php`
- `DocumentService`, `TestController` — использование `public_path`

## Риски
- Потеря данных при сбое сервера.
- Невозможность масштабирования (кластеры, CDN).
- Ограничения по дисковому пространству.

## Рекомендации
1. Настроить S3/MinIO как основное хранилище документов и бэкапов.
2. Разделить диски: `public` для статики, `private` для документов.
3. Подключить CDN для раздачи статических ресурсов.
