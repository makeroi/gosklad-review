# 24. Строгие красные флаги

## Результат
- **Статус:** ❌ Выявлены нарушения
- **Severity:** S0
- **Краткое заключение:** большинство красных флагов присутствуют в проекте и требуют немедленных действий.

## Список флагов
- [x] Секреты в репозитории, `APP_DEBUG=true` на prod *(02.1, 02.2, 05.1)*
- [x] Нет бэкапов или проверки восстановления *(19.3)*
- [x] Отсутствуют очереди при тяжёлых задачах *(10.1)*
- [ ] Устаревшие версии PHP/Laravel *(планируется проверить в 3.1)*
- [x] Нет ограничений БД и массовые N+1 *(07.*, 06.2)*
- [x] Нет мониторинга по 5xx и очередям *(09.*, 10.1)*
- [x] Публичные незакрытые эндпоинты или вебхуки без подписи *(05.1, 12.3, 18.2)*

## Рекомендации
- Закрыть все отмеченные флаги в рамках Фазы 1 дорожной карты.
- Завести отдельный борд для отслеживания устранения S0/S1.
